# TrackMotion Privacy Policy

This repository contains the privacy policy website for the TrackMotion mobile application.

## Overview

The website provides a comprehensive privacy policy that explains:
- What information we collect
- How we use the information
- How we share and protect user data
- User privacy rights
- Contact information

## Viewing the Website

To view the privacy policy website locally:

1. Clone this repository
2. Open `index.html` in your web browser

## Deployment

This website can be deployed to any static hosting service such as:
- GitHub Pages
- Netlify
- Vercel
- AWS S3
- Any web server

## Files

- `index.html` - Main HTML page containing the privacy policy
- `styles.css` - Stylesheet for the privacy policy page
- `README.md` - This file
